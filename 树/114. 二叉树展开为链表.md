## [114. 二叉树展开为链表](https://leetcode.cn/problems/flatten-binary-tree-to-linked-list/submissions/)

1. 时间复杂度o(n) 空间复杂度o(n)
```ts
function flatten(root: TreeNode | null): void {
    if (!root) return null;
    flatten(root.left);
    flatten(root.right);
    let left = root.left;
    let right = root.right;
    root.left = null;
    root.right = left;
    let temp = root;
    while (temp.right) {
        temp = temp.right;
    }
    temp.right = right;
};
```
