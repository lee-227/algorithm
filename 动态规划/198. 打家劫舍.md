## [198. 打家劫舍](https://leetcode.cn/problems/house-robber/)

1. 动态规划
```ts
var rob = function (nums) {
    let n = nums.length;
    const dp = new Array(n + 1).fill(0);
    dp[0] = 0;
    dp[1] = nums[0];
    for (let i = 1; i < n; i++) {
        dp[i + 1] = Math.max(dp[i], dp[i - 1] + nums[i])
    }
    return dp[n]
};
```
